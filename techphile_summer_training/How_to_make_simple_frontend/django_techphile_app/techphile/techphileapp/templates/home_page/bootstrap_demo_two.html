This page demonstrates how to use bootstrap classes

  • mx it gives margin in x axis both at th starting and at the end of the element
  • my it gives margin at the top and bottom of the element
  • mb it gives margin at the bottom of the element
  • mt it gives margin at the top of the element
  • fs it gives the font its size
  • p it gives padding in all sides top, bottom, start and end
  • ps-5 padding start gives padding in the start
  • pe-5 padding end gives padding in the end
  • pt-5 padding top gives padding in the top
  • pb-5 padding bottom gives padding in the bottom
  • Note: each class wheather its m, p, fs class and all of its sub classes have value between 0~5
