This page is a demonstration of bootstrap

This page demonstrates how to use bootstrap classes

  • How to use row and column class to partition the screen
  • If you want to partition the screen vertically then make row your parent and then use column to partition the screen vertically. Each Row have 12 columns
  • If you want to partition the screen horizontally then make the column your parent and then use row to partition the screen horizontally. Each column have 12 rows
  • A column cannot have another column as a child
  • A row cannot have another row as a child
  • you have to use row and column alternatively in order to get the desired partition on a page
  • row and column classes in bootstrap makes partitioning of the page very easy
